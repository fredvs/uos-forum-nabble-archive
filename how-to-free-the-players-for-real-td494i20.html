  { TForm1 }

  TForm1 = class(TForm)

   procedure TimerFadeTimer(Sender: TObject);

implementation

...

 procedure Tform1.MyEndProc();     // ----> add this
begin
// add your code
end;

______________________________

And with your player, define uos_EndProc like this:

 uos_EndProc(nPlayer, @MyEndProc);
